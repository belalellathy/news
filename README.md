📰 News App
A simple Flutter news application demonstrating API integration, responsive UI, search, theme switching, and modern state management. This project is suitable as a demo, a learning resource, or a foundation for a more advanced news app.
📌 Features
🔎 Fetches real-time news from a REST API (e.g., NewsAPI.org)
🗂️ Browse news by categories (e.g., Business, Sports, Science)
🔍 Search functionality to find specific articles
🌙 Light & Dark theme support using system preferences
📱 Smooth, responsive UI for Android & iOS
🧠 Provider-based state management
⚡ Async UI updates using FutureBuilder
🧰 Tech Stack
Flutter & Dart – UI & app logic
HTTP package – REST API requests
Provider – State management
FutureBuilder – Async UI rendering
