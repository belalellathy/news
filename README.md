# News App

A simple Flutter news application showcasing API integration, modern state management, theme switching, and search functionality. Ideal as a demo or proof‑of‑concept, it includes key features you’d expect in a production news app.
 Features
Fetches real-time news from a REST API (e.g. News.org API)
Browse by categories: Business, Sports, Science
Full search capability across fetched articles
Light & Dark theme depends on device system theme
Smooth, responsive UI on Android & iOS
 Technical Stack
Flutter & Dart — core UI and logic
HTTP  — for REST API requests
State Management: Provider 
FutureBuilder : to manage async UI updates
